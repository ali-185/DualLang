Readme

This program is for converting an ePub file into a dual text ePub file.

To run:

Double click dual_lang_gui.exe

Required Microsoft Visual C++ 2008 Redistributable Package: http://www.microsoft.com/downloads/en/details.aspx?FamilyID=9b2da534-3e03-4391-8a4d-074b9f2bc1bf&displaylang=en

To create the exe:

>python setup.py install
>python setup.py py2exe

Reference: http://www.py2exe.org/index.cgi/Tutorial
